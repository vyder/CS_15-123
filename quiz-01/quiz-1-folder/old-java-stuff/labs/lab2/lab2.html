Background:

This lab is a re-make of Lab1. The main tester program I will use is the same one I used to test your Leb1. You should rename it to Lab2 and reuse if for testing of our Lab2 version of your Graph class.

The Graph class is where you will make changes.  You will store your graph in an array g[] of Edge pointers.  An Edge object is just a small class that holds a destination node #, a weight value, and a pointer to the next Edge.  In your Graph class you will replace your int g[][] with Edge g[].  

Each element of g[] will be a pointer to an Edge object. Thus each element of g[] is the head of a linked list. Each linked list headed by g[i] represents a list of edges leaving node i.

Because the edge information is stored in linked lists, the implementation of the graph's Constructor, readGraphFile, AddEdge(), inDegree() and outDegree() and print() will change.  When rewriting the Graph methods that are changing - you should refer to the Linked List demo code given in class.

All the public degree methods methods in Graph.java (the ones called from main) will not change for Lab2 . They will work exactly as written in Lab1.
