Lab 6: Applets, Threads and simple Animation

Due Date: Tue 11 Apr 2006 by 11:59pm

Background:

Lab6 is an enhancement of the RacingBalls applet. We enhance our Applet by adding threading to replace the hardcoded delay loop. We also use a JSlider control to allow the user to adjust the speed at which the balls advance across the screen.

The Assignment

You are given no starter files other than what you can find on our course website

  • Applet must have 3 buttons Reset, Start & Clear.
  • Balls must be Red Green and Blue
  • Balls must start at 2/5, 3/5, 4/5 down the left side of the applet.
  • Balls must be 50 pixels in diameter.
  • You must erase the previous position of the ball - no blur across screen.
  • Threading must be used to control the speed of the balls
  • JScrollBar as demoed in todays lecture notes (or JSlider - see API) control must be present to allow user to change ball speed while balls are racing.
  • The minimum delay is 1 millisecond, the max is 1 second.
  • You must turn in TWO source files - a JApplet named ThreadedRace.java and a .html file for it.
